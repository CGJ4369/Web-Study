1.元素显示模式

元素显示模式就是元素(标签)以什么方式进行显示，比如div标签自己占一行，一行可以放多个span标签.

元素一般分为块元素和行内元素两只类型

1.1块元素

常见的块元素有h1~h6、p、div、ul、ol、li等

div标签是最典型的块元素

块元素的特点有：
  1. 独占一行
  2. 高度宽度以及外边距都可以控制
  3. 宽度默认是容器(父级宽度)的100%
  4. 是一个容器及盒子，里面可以放行内或者块级元素
注意：文字类的元素内不能放块级元素
如p标签，h标签

1.2行内元素

常见的行内元素有a标签，strong标签，span标签等

span标签是最典型的行内元素

行内元素的特点有：
  1. 相邻行内元素在一行上，一行可以显示多个
  2. 高宽直接设置是无效的
  3. 默认宽度就是它本身内容的宽度
  4. 行内元素只能容纳文本或其他行内元素
注意：
  • 链接里面不能再放链接
  • 特殊情况链接a里面可以放块级元素，但是给a转换下一块级最安全

1.3行内块元素

行内元素有几个特殊标签img标签、input标签、td标签它们同时具有块元素和行内元素的特点

行内块元素的特点：
  1. 和相邻行内元素(行内块)在一行上，但是他们之间会有空白间隙。一行可以显示多个(行内元素的特点)
  2. 默认宽度就是它本身内容的宽度(行内元素特点)
  3. 高度，行高，外边距以及内边距都可以控制(块级元素特点)

1.4元素显示模式转换

特许情况下，我们需要元素模式的转换，简单理解：一个模式的元素需要另一种模式的特性

比如想要增加链接a的触发范围
