5.背景图像固定

6.背景复合写法

为了简化背景属性的代码,我们可以讲这些属性合并简写在同一个属性background中

简写属性时没有特定的书写顺序,一般习惯约定顺序为：

background:背景颜色 背景图片地址 背景平铺 背景图像滚动 背景图片位置
